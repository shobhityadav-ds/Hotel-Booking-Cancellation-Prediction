# Data Folder

This folder contains the dataset used in this project.

## Dataset

**File**

hotel_bookings.csv

**Description**

The dataset contains hotel booking records collected from City Hotels and Resort Hotels.

Target Variable:

- **is_canceled**
  - 0 → Booking Not Cancelled
  - 1 → Booking Cancelled

Dataset includes customer demographics, booking details, hotel information, lead time, market segment, deposit type, ADR, and other booking-related features.

Source:

https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand
