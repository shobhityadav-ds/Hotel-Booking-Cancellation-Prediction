# Images Folder

This folder contains the visualizations generated during Exploratory Data Analysis (EDA) and Machine Learning model evaluation.

## Available Images

### Correlation Heatmap
Displays the correlation between important numerical features in the dataset.

### Logistic Regression Confusion Matrix
Shows the prediction performance of the Logistic Regression classifier.

### Decision Tree Confusion Matrix
Shows the classification results of the Decision Tree model.

### Random Forest Confusion Matrix
Displays the prediction performance of the Random Forest classifier.

### KNN Confusion Matrix
Shows the prediction performance of the K-Nearest Neighbors model.

### ROC Curve Comparison
Compares the ROC curves and AUC scores of all machine learning models.

### Top 10 Important Features
Displays the most influential features identified by the Random Forest model.

These visualizations support the analysis and evaluation presented in the project notebook.
